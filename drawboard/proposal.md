# Final project proposal

## 遊戲名稱：方塊遊戲

## 遊戲簡介

- 遊戲玩法
  以俄羅斯方塊為核心，每次消行都會對於路上的敵人造成傷害，敵人攻擊的時候你會收到垃圾排。遊戲目前只有一關，分為普通敵人跟BOSS戰，BOSS會有獨特的攻擊方式。

- 傷害計算

  - [參考Tetris Guideline](https://harddrop.com/wiki/Tetris_Guideline)

  - Combo table  
    Multiplier
  - Spins  
    All spin+
  - Garbage cancellation  
    +1 damage
  - Back-to-Back cancellation  
    +1 damage if Back-to-Back

垃圾排是灰色方塊，從底下往上長。每橫排都有一個缺口，只要補滿缺口就能消排。

- Mechanics

  - SRS+ kick
  - 7 bag system
  - Configurable controls
